# Stellar Burgers API Testing

## Описание проекта

Stellar Burgers - веб-приложение для сборки и оформления заказов бургеров с регистрацией и отслеживанием статуса заказов.

### Цель проекта

Автоматизация тестирования API: проверка основных сценариев

### Тип тестирования

API

### Реализованные задачи

- Разработаны автотесты для регистрации и логина пользователя.
- Разработаны автотесты для создания заказа.
- Организована генерация динамических тестовых данных.
- Протестированы позитивные и негативные сценарии.
- Добавлены аннотации для формирования отчёта в Allure.

## Структура проекта

```text
project/
    ├─ main/
    │   └─ java/
    │       ├─ config/         # конфигурация RestAssured
    │       ├─ model/          # POJO классы 
    │       └─ steps/          # классы шагов реализации
    └─ test/
        └─ java/
            ├─ BaseTest.java      # базовый тестовый класс
            └─ ...                # тесты
```

## Технологии

- Java 11
- Maven
- JUnit 4
- RestAssured
- Jackson
- Datafaker
- Allure

## Конфигурация и запуск

### Запуск тестов

*mvn clean test*

### Формирование отчёта

*mvn allure:serve*
